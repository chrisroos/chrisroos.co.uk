Getting my Webby powered website running on Mac OS X El Capitan

This is what I needed to do to get Webby powered website working on my laptop.

$ cd ~/Code/github/chrisroos.co.uk

$ cat .rbenv-version
1.8.7-p352

$ rbenv install 1.8.7-p352
