Day notes - Mon 18th Oct 2010

Started using a shared keychain file (stored in shared dropbox instead of a password protected dmg containing a text file full of passwords.

Worked with Tom to start migrating the hosting of an app from linode to AWS.

O2 meeting with Kalv.

Worked with James A to get a performance testing story accepted.

Played around with the security command line tool to extract a private key from a secure note in the keychain. You’ll need something like this if you want to do the same.

  $ security find-generic-password -g -l "my-label" 2>&1 1>/dev/null | ruby -e 'puts STDIN.read[/-----BEGIN PRIVATE KEY-----.*-----END PRIVATE KEY-----/].gsub(/\\012/, "\n")' > ~/path-to-key
